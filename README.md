# Sistema de Gestión de Vacaciones - Chatbot

## Descripción

Programa que consiste en la simulación de un chatbot para la gestión de solicitudes de vacaciones de empleados. El sistema valida la identidad del empleado mediante su legajo, verifica la disponibilidad de días de vacaciones mediante el archivo CSV y aprueba o rechaza la solicitud teniendo en cuenta estos datos.

El proceso fue modelado utilizando BPMN 2.0 y posteriormente implementado en Python.

## Funcionalidades

* Validación de legajo.
* Consulta de empleados desde archivo CSV.
* Verificación de días disponibles.
* Manejo de errores de entrada.
* Control de intentos inválidos.
* Derivación a Recursos Humanos ante situaciones excepcionales.

## Ejecución

1. Descargar o clonar el repositorio.
2. Verificar que los archivos CSV se encuentren en la misma carpeta que el programa.
3. Ejecutar.
